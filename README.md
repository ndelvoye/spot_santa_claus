Spot Santa Claus
============================================================
TP Réseaux & Flux multimédia - ENSSAT 2020

Technologies
------------
* ExpressJS
* SocketIO
* Leaflet

Run
---
1. `npm install`
2. `npm start`
3. Connect to `http://localhost:3000/`

Live demo
---------
https://spot-santa-claus.herokuapp.com/
